## Citation Format

For academic consistency, use the following citation format throughout the module:

### In-text citations:
- For documentation: (NVIDIA Isaac ROS Documentation, 2023)
- For research papers: (NVIDIA Research Team, 2022)

### Reference list format:
For documentation:
```
NVIDIA Isaac ROS Documentation. (2023). Retrieved from https://nvidia-isaac-ros.github.io/
```

For research papers:
```
NVIDIA Research Team. (2022). Hardware Accelerated Visual SLAM for Real-time Applications. IEEE Robotics and Automation Letters.
```

## Verification Guidelines

All technical claims in this module must be verified against these official sources. If a claim cannot be verified with an official NVIDIA document or peer-reviewed research paper, it should be marked with a "NEEDS VERIFICATION" tag and researched further before publication.

## Best Practices for Citations

1. Always cite the most recent version of the documentation
2. Include version numbers when referencing specific Isaac Sim or Isaac ROS releases
3. Link to the exact page when possible, not just the top-level documentation site
4. When referencing code examples, cite the specific GitHub repository and commit if relevant
5. Verify that all links are functional and lead to current documentation
